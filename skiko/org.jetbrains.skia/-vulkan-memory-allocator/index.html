VulkanMemoryAllocator

abstract class VulkanMemoryAllocator

Controls how Skia allocates Vulkan device memory.

You may pass an implementation of this class to DirectContext.makeVulkan. When omitted, a built-in per-allocation allocator is used that is always functional, however, it does not do any sub-allocations.
